# Week 1 Notes — IT Fundamentals

## Computer Hardware
- CPU — processes all instructions
- RAM — temporary memory, lost when power off
- Storage — HDD/SSD, permanent memory
- Motherboard — connects all components together

## Operating Systems
- Windows, macOS, Linux — different OS types
- Linux is used in cybersecurity because it is open source and flexible

## File System
- Directories and paths — how files are organised
- Linux uses / as the root directory

## Linux Terminal Basics
- pwd — shows current directory
- ls — lists files in current directory
- cd — change directory
- mkdir — make a new folder
- rm — remove a file
- cp — copy a file

## Networking Intro
- IP address — unique address for every device on a network
- MAC address — physical address burned into your network card

## OSI Model — 7 Layers
- Layer 7 — Application (what you see, HTTP, FTP)
- Layer 6 — Presentation (encryption, formatting)
- Layer 5 — Session (manages connections)
- Layer 4 — Transport (TCP/UDP, data delivery)
- Layer 3 — Network (IP addresses, routing)
- Layer 2 — Data Link (MAC addresses)
- Layer 1 — Physical (cables, signals)
